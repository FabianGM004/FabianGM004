## 📁 Key Projects & Lab Work

### 🖥️ Proof of Concept – Container-Based Development Environments for Existing Active Directory Classrooms

*Advanced Technician in Network Computer Systems Administration (ASIR)*

A technical Proof of Concept demonstrating how Kubernetes-based development environments can be integrated into existing Windows classrooms managed by Active Directory without replacing the current infrastructure.

* **The Mission (Use Case):** Design and validate an architecture that enables educational institutions to adopt modern containerized development environments while preserving their existing Windows infrastructure, Active Directory services, and classroom management model.

* **Core Concept:** Design and implementation of a lightweight Kubernetes (k3s) cluster running on WSL2, where the teacher workstation operates as the control plane and student workstations join the cluster as worker nodes, contributing their own computing resources.

* **Key Technologies:** Kubernetes (k3s), Windows 11, WSL2, Active Directory Domain Services (AD DS), PowerShell, Bash, Portainer, VMware Workstation and VS Code Server.

* **Key Engineering Challenges:**

  * Investigated WSL2 startup limitations and manual initialization requirements.
  * Solved Windows-to-WSL2 networking issues using Windows port forwarding.
  * Evaluated WSL2 Mirrored Networking and documented its limitations within the testing environment.
  * Implemented persistent student storage using Windows bind mounts.
  * Automated classroom deployment using PowerShell and Bash scripts.
  * Documented the architecture, deployment process, implementation challenges, results and future improvements in a complete technical repository.
